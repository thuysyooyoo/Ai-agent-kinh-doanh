### 6.2 Scripts xử lý Objection phổ biến

#### OBJECTION 1: "Giá bên em cao quá"

**ACKNOWLEDGE:**
```
"Em hiểu giá là yếu tố quan trọng. Nhiều khách hàng của ERK cũng có
cùng quan tâm này ban đầu."
```

**CLARIFY:**
```
"Anh/chị có thể chia sẻ so với mức nào thấy cao không?
- So với đối tác hiện tại?
- So với bên khác vừa quote?
- Hay so với budget anh/chị có?"
```

**RESPOND (Scenario 1: So với đối tác hiện tại):**
```
"Em hiểu. Nhưng nếu anh/chị xét tổng chi phí (total cost):
• Đối tác hiện tại: Giá có thể thấp hơn nhưng có phát sinh phí không?
  - Phí HQ riêng?
  - Phí lưu kho?
  - Phí handling?
• ERK: Giá all-in-one, không phát sinh
• Plus: OTD 95%+ - giảm rủi ro stock-out, production delay

Thực tế nhiều khách hàng chuyển sang ERK后发现 total cost thấp hơn."
```

**RESPOND (Scenario 2: So với bên khác):**
```
"Em hiểu. Khi so sánh, anh/chị có check kỹ không:
• Có bao trọn HQ không hay phí HQ riêng?
• Có bao insurance không?
• Có phí fuel surcharge không?
• OTD của họ bao nhiêu?

Nhiều khi quote thấp nhưng phát sinh phí, tổng lại cao hơn.
Em có thể breakdown chi tiết cho anh/chị xem."
```

**RESPOND (Scenario 3: Budget constraint):**
```
"Em hiểu budget là quan trọng. Em có vài options:
• Adjust service level (ví dụ đường biển thay vì bộ)
• Volume commitment để có better rate
• Payment terms linh hoạt hơn

Em có thể work với anh/chị để tìm solution phù hợp budget."
```

**CHECK:**
```
"Ngoài giá ra, yếu tố nào quan trọng nhất với anh/chị?"
```

## 10. BEST PRACTICES

### 10.1 Time Management

| Tip | Chi tiết |
|-----|----------|
| **Block time** | Book time cho tasks lớn (prospecting, follow-ups) |
| **Batch activities** | Gom việc giống nhau (calls, emails) |
| **Prioritize** | Focus on high-value opportunities first |
| **Plan ahead** | Chuẩn bị trước cho tuần/ngày tiếp theo |
| **Avoid distractions** | Tắt notifications khi làm việc tập trung |

**Daily Schedule mẫu:**
```
8:00-8:30    Check email, Zalo, plan day
8:30-10:00   Prospecting calls (fresh energy)
10:00-11:00  Follow-up calls
11:00-12:00   Admin, CRM update
13:30-15:00  Meetings / Deep work
15:00-16:00   Follow-ups, emails
16:00-17:00   Admin, wrap up
17:00-17:30   Plan tomorrow
```

### 10.2 Territory Management

| Tip | Chi tiết |
|-----|----------|
| **Segment by geography** | Phân chia theo khu vực |
| **Segment by industry** | Phân chia theo ngành |
| **Balance portfolio** | Mix khách lớn + nhỏ |
| **Focus on potential** | Ưu tiên khách có tiềm năng |
| **Review regularly** | Review territory quarterly |

### 10.3 Key Account Management

**Định nghĩa Key Account:**
- Volume ≥ 200tr/tháng
- Strategic importance
- Growth potential
- Referral source

**Key Account activities:**
- Weekly check-in
- Quarterly business review
- Priority support
- Custom pricing considerations
- Executive relationship
